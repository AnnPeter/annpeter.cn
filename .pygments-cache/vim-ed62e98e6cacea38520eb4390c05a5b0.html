$ sudo apt-get install tftp-hpa(tftp-hpa是客户端，如果不许要可以不安装)</p>

<p>//配置tftp服务器
$ sudo vim /etc/default/tftpd-hpa
将原来的内容修改为
  TFTP_USERNAME=&ldquo;tftp&rdquo;
  TFTP_ADDRESS=&ldquo;0.0.0.0:69&rdquo;
  TFTP_DIRECTORY=&ldquo;tftp根目录&rdquo; //服务器目录,需要设置权限为777
  TFTP_OPTIONS=&ldquo;-l -c -s&rdquo;</p>

<p> //重启tftp服务器
 $ sudo service tftpd-hpa restart
