本文主要介绍我常用的AndroidStudio快捷键。

<!-- more -->

* 格式整理			Cmd + Option + L
* 清除无效引用包		Option + Control + O
* 导入包		Option + Enter
* 选择父类方法进行写  	Ctrl + O
* 构造函数、get和set方法 Ctrl + N
* 各种包裹语句   Cmd+Option+T
* 代码格式化 	OPTION + CMD + L
* 查看类的继承关系  Ctrl + H
